# Claude Admin Utilities

Admin-only utility bundle for power users managing Claude/Cowork setup and recurring workflows.

## Included skills

- `memory-consolidation`
- `setup-cowork`
- `schedule`
- `create-plugin`
- `skill-lookup`

## Provisioning note

Provision this plugin only to users whose work matches the workflow. Avoid enabling every plugin globally; narrower bundles reduce accidental triggers and connector confusion.
