# Document Production

Optional file-production bundle for users who create and edit formal documents, decks, PDFs, and workbooks.

## Included skills

- `doc-coauthoring`
- `word-documents`
- `pdf-documents`
- `powerpoint-decks`
- `excel-workbooks`
- `theme-factory`

## Provisioning note

Provision this plugin only to users whose work matches the workflow. Avoid enabling every plugin globally; narrower bundles reduce accidental triggers and connector confusion.
