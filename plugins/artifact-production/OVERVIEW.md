# Artifact Production

Overincluded optional production tools for rich visual, web, and creative artifacts.

## Included skills

- `artifacts-builder`
- `web-artifacts-builder`
- `canvas-design`
- `theme-factory`
- `algorithmic-art`
- `slack-gif-creator`
- `visual-design-foundations`
- `image-enhancer`

## Provisioning note

Provision this plugin only to users whose work matches the workflow. Avoid enabling every plugin globally; narrower bundles reduce accidental triggers and connector confusion.
