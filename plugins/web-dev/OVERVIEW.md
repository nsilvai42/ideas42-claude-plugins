# Figma Web Dev

Design-to-development support for teams with Figma or frontend implementation work.

## Included skills

- `figma-implement-design`
- `figma-code-connect`
- `figma-design-system-rules`
- `react-performance`
- `web-ui-review`
- `visual-design-foundations`
- `image-enhancer`
- `ui-ux-design-review`

## Provisioning note

Provision this plugin only to users whose work matches the workflow. Avoid enabling every plugin globally; narrower bundles reduce accidental triggers and connector confusion.
